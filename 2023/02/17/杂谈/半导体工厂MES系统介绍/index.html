半导体车间MES系统介绍

Bo Zhang 2023-02-17 23:05:19
Categories： Tags：

1. 什么是 MES 系统

MES 是 Manufacturing Execution System 的简称，即制造执行系统。MES 通过信息的传递，对从订单下达到产品完成的整个生产过程进行优化管理，对工厂发生的实时事件及时作出相应的反应和报告，并使用当前的准确数据对进行指导和处理。

2. MES 系统的应用对象

MES 系统面向车间，重点关注如何高效地执行制造以及处理订单。MES 系统面向的对象可以细分为以下几部分：

  • 人
  • 机器
  • 物料（Material）
  • 方法
  • 环境
  • 测试

以半导体车间为例，可以具体分为：

  • 测试人员（Test Operator）
  • 测试仪器（Test Equipment）
  • 被测器件（Device under Test）
  • 测试规格（Test Specification）
  • 测试程序（Test Program）
  • 测试结果（Test Results）

3. MES 系统的组成与功能

MES 系统主要分为四大类：

  • 生产管理
    • 计划排程
    • 生产调度
    • 工艺管理
  • 物料管理
    • 批次管理
    • 包装管理
    • 配送
  • 质量管理
    • 品质管理
    • 质量控制分析
    • Fail 管理
  • 维护管理
    • 数据采集
    • 状态实时监控
    • 维修过程管理
    • 定期维护

4. 为什么用 MES 系统

MES 系统是现代智能工厂不可或缺的一环。它摒弃了原始的人工记录报表或者 Excel 表格储存信息的方式，而使用一种全新的实时系统来检测数据。使用条形码对每一个仪器、产品和装配组件进行追踪，确保及时获取数据和反应问题。还是以半导体车间的 Final Test 为例，每条产线上都有特定的测试仪器，用以特定测试的执行，如：

  • 静态测试
  • 动态测试
  • 绝缘测试
  • BIPSS

对于每一种产品，都应该有相应的配套设备：

  • 测试板（Adapter）
  • 门极电阻 / 钳位插头（RG- / Clamping Plug）
  • 工件载体（Workpiece Carrier）
  • 测试结束后用于装器件的托盘（Tray）

仪器和产品之间需要特定的测试程序进行连接，被称为产品的配方（Recipe）。
